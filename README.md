# Голосовой ассистент для Android

Умеет две вещи:
- **Звонить** по имени из контактов или по номеру
- **Отвечать на любые вопросы** через GigaChat (Сбер, бесплатно)

## Быстрый старт

```
1. Дважды кликни  ЗАПУСТИТЬ_НАСТРОЙКУ.bat
   (вставь ключ GigaChat когда спросит)

2. После настройки мастер запустит сервер автоматически

3. Открой android/ в Android Studio → запусти на телефоне
```

## Голосовые команды

| Скажи | Что произойдёт |
|-------|---------------|
| «Позвони Маше» | Звонит контакту «Маша» |
| «Набери +79001234567» | Звонит на номер |
| «Какая погода в Москве?» | Отвечает GigaChat |
| «Что такое черная дыра?» | Отвечает GigaChat |
| Любой вопрос | GigaChat отвечает и озвучивает |

## Структура

```
voice-assistant/
├── ЗАПУСТИТЬ_НАСТРОЙКУ.bat   ← один раз
├── ЗАПУСТИТЬ_СЕРВЕР.bat      ← каждый раз
├── backend/
│   ├── main.py               ← один роут: POST /ask
│   ├── ai_service.py         ← GigaChat API
│   └── requirements.txt
└── android/
    └── ...
```

## Настройка Android (реальный телефон)

В `BackendApiClient.kt` замени:
```kotlin
var baseUrl: String = "http://10.0.2.2:8000/"
// на:
var baseUrl: String = "http://192.168.X.X:8000/"  // IP твоего компьютера
```
